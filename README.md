# medapp

Proyecto Flutter inicial generado manualmente por el asistente.

Instrucciones rápidas:

1. Verifica que Flutter esté instalado y funcional:

```bash
flutter doctor
```

2. Desde `c:\\Users\\2002s\\Documents\\medapp` instala dependencias:

```bash
flutter pub get
```

3. Si deseas generar las carpetas nativas completas (android/ios), ejecuta desde el directorio del proyecto:

```bash
flutter create .
```

4. Ejecuta la app en un emulador o dispositivo:

```bash
flutter run
```

Notas:
- Para compilar en iOS se requiere macOS + Xcode.
- Cambiar el `applicationId` / bundle identifier: editar los archivos nativos generados en `android/` e `ios/`.
